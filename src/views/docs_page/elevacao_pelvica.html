Elevacao Pelvica

  1. Sente-se no chão com um banco atrás de você. Coloque a barra sobre as pernas logo acima dos quadris.
  2. Incline-se contra o banco para que seus ombros fiquem apoiados nele, estique os braços para os lados usando o banco como apoio.
  3. Suba o peso com os pés e estendendo os quadris para cima. Apoie o peso com os ombros e os pés.
  4. Estenda lentamente o máximo que puder e, em seguida, retorne lentamente à posição inicial.
