O administrador vai ter acesso para a parte do CRUD das tabelas.
Um utilizador normal não deve ter permissões para entrar na parte do CRUD das tabelas.

Apenas o administrador pode visualizar a rota /admin.

Existem dois tipos de utilizadores, administrador e utilizador comum.

Para criar conta utilizador, o utilizador vai preencher os seguintes campos:

- Nome Completo
- Username
- Email
- País
- Cidade
- Password


Ao fazer login, é so necessário o username e password.
